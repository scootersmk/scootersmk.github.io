Open Source Software

Having been a Linux/Unix systems administrator my entire career I have benefitted heavily from free and open source software. Therefore, I have always felt its important for me to make contributions when I can to give back to the community. What follows is a list of some of my contributions big and small that I have worked on over the years. I hope I can continue to find time to keep adding to this list.

My Contributions

Pam_radius

  • Added support for the prompt attribute in the radius protocol

Testinfra

  • Added support for testing for iptables rules

Repowatch

  • Fix crash bug when deleting branch
  • Restart on crash with systemd

Upstream_sync

  • Mirror with ftp and support aarch64
  • Mirror source rpms

Rkrellm

  • Project I wrote from scratch mainly to get experience using automake/autoconf/etc
